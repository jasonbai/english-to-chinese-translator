# IELTS Vocabulary Levels Reference

This document defines the vocabulary grading standards for IELTS Band 6, 7, and 8 levels.

## Band Level Definitions

### Band 6 (CEFR B2-C1)
- **Definition**: Competent user with generally effective command of language
- **Vocabulary Range**: Can use a range of vocabulary for a variety of topics
- **Characteristics**:
  - Some less common and idiomatic vocabulary
  - Awareness of style and collocation
  - Occasional errors in word choice

### Band 7 (CEFR C1)
- **Definition**: Good user with operational command of language
- **Vocabulary Range**: Uses less common lexical items skillfully
- **Characteristics**:
  - Precise vocabulary for specific meanings
  - Good awareness of collocation and style
  - Rare errors in word choice

### Band 8 (CEFR C1-C2)
- **Definition**: Very good user with fully operational command
- **Vocabulary Range**: Uses a wide range of vocabulary flexibly and precisely
- **Characteristics**:
  - Sophisticated control of lexical features
  - Skillfully uses uncommon and idiomatic items
  - Rare minor errors

## Collocation Patterns (Band 7+)

### Verb + Noun Collocations
- **conduct** research / an experiment / a survey
- **draw** a conclusion / a distinction
- **pose** a threat / a challenge / a risk
- **reach** a consensus / an agreement / a verdict
- **shed light on** an issue / a problem

### Adjective + Noun Collocations
- **compelling** evidence / argument / reason
- **daunting** challenge / task / prospect
- **pressing** issue / concern / matter
- **stark** contrast / difference / reality
- **underlying** cause / factor / reason

### Adverb + Adjective Collocations
- **highly** controversial / influential / significant
- **increasingly** apparent / evident / important
- **profoundly** affected / influenced / different
- **widely** acknowledged / believed / recognized
- **thoroughly** investigated / examined / reviewed

## Do NOT Highlight (Common Mistakes)

These words are too common for Band 6+ highlighting:

### Basic Academic Words (Band 5)
- analyze, approach, benefit, concept, create, data
- develop, effect, environment, factor, function, impact
- issue, method, process, result, significant, structure
- theory, therefore, various

### Common Transition Words
- however, therefore, moreover, nevertheless, consequently
- furthermore, additionally, alternatively, ultimately

### Basic Verbs
- show, give, make, take, use, get, have, know, see
